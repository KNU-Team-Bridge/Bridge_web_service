# BRIDGE : 청각장애인과 비장애인 화상회의 플랫폼

<br>

## 1. 프로젝트 개요
청각장애인의 비율이 빠르게 증가함에 따라 **청각장애인들의 사회 활동을 돕는 서비스**가 필요하다고 생각했습니다. 코로나 시대에 맞춰 영상 회의 서비스에 주목했습니다.

청각장애인과 비장애인 화상 회의 플랫폼 ‘BRIDGE’를 만들게 되었습니다.
